# 🛒 Retail Sales Analysis and Prediction

## 📌 Project Overview

This project focuses on analyzing retail sales data from a coffee shop and building machine learning models to predict revenue. The project combines data cleaning, exploratory data analysis (EDA), visualization, and predictive modeling to generate meaningful business insights.

---

## 🎯 Objectives

* Clean and preprocess retail sales data
* Perform Exploratory Data Analysis (EDA)
* Identify sales patterns and trends
* Visualize important business metrics
* Build machine learning models for revenue prediction
* Compare model performance and draw conclusions

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn
* Jupyter Notebook

---

## 📂 Dataset

**Dataset Name:** Coffee Shop Sales Dataset

### Features

* Transaction ID

* Transaction Time
* Transaction Quantity
* Store Location
* Product Category
* Product Type
* Product Detail
* Unit Price

## 🔍 Key Findings

* Store location significantly affects revenue generation.
* Coffee products contribute the highest sales.
* Revenue strongly depends on transaction quantity and unit price.
* Peak sales occur during specific business hours.
* Machine learning models successfully predict revenue with high accuracy.

## 💡 Business Recommendations

* Focus inventory on high-performing products.
* Increase staffing during peak sales hours.
* Use sales trends for demand forecasting.
